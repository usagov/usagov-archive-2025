Los sitios web oficiales usan .gov
Un sitio web .gov pertenece a una organización oficial del Gobierno de Estados Unidos.

Los sitios web seguros .gov usan HTTPS
Un candado ( ) o https:// significa que usted se conectó de forma segura a un sitio web .gov. Comparta información sensible sólo en sitios web oficiales y seguros.

Directorio de oficinas de elecciones estatales y locales

Encuentre su oficina de elecciones estatal o local para registrarse para votar y obtener ayuda con otros asuntos relacionados con las elecciones.

Seleccione Español o Spanish en la esquina superior derecha o al final de la página de su estado si es necesario. Si su estado no cuenta con página web en español, por favor llame al número incluido en la página en inglés para asistencia en su idioma.
