Programa de Cupones para Alimentos (SNAP)

La misión del Programa de Cupones para Alimentos (SNAP, sigla en inglés) es eliminar el hambre y mejorar la nutrición y la salud. El programa ayuda a las familias de bajos ingresos a comprar la comida necesaria para una dieta nutritiva.
